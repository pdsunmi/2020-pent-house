# PENT HOUSE
> 공공데이터 기반 아파트/연립다세대 실거래가를 검색할 수 있는 웹서비스 PENT HOUSE를 소개합니다. Spring Boot와 Vue.js를 이용하였으며 **RESTful**기반으로 **SPA**를 구현하였습니다.
> 
> 개발 기간 : 약 10일
> 개발 인원 : 2명
> 적용 기술 : Spring Boot / Vue.js /Node.js / MyBatis / Google Maps API / BootStrap
